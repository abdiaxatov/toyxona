# Qog'oz Menyu Skaneri - Qo'llanma

## Restoran Yaratish

1. Super Admin panelga kiring:
   ```
   http://localhost:3000/admin/super-admin
   ```

2. "Yangi Restoran" tugmasini bosing

3. **Asosiy Ma'lumotlar:**
   - Restoran nomi: `Test Restaurant`
   - Slug: `vcbnn`

4. **Menyu Dizayni:**
   - "Skan" variantini tanlang

5. **Qog'oz Menyu Rasmlari:**
   - "Sahifalarni Yuklash" tugmasini bosing
   - Menyu rasmlarini tanlang (bir nechta rasm)
   - Rasmlar avtomatik yuklanadilar

6. "Bitirish va Yaratish" tugmasini bosing

## Natijani Ko'rish

Brauzerda quyidagi manzilga o'ting:
```
http://localhost:3000/vcbnn
```
